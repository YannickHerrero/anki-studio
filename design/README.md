# Card design

[`japanese-sentence-card.html`](./japanese-sentence-card.html) is a self-contained preview of the Anki note type used by Anki Studio. Open it in a browser to see the front and back of the card on both mobile and desktop, in light and dark mode.

The actual Anki templates extracted from this design live in [`../server/src/anki/`](../server/src/anki/).
